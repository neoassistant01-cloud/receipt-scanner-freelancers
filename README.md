# ReceiptSnap

Receipt scanning app for freelancers with AI categorization and accounting software export.

## Features

- **📸 Scan Receipts** — Camera capture or gallery import
- **🧠 Auto-Categorize** — AI suggests IRS Schedule C categories (Advertising, Car & Truck, Meals, Travel, etc.)
- **📊 Monthly Reports** — View spending by category with visual bar charts
- **📤 Export** — Download CSV or export to QuickBooks/Xero

## Usage

1. Open `dist/index.html` in a browser
2. Create an account or sign in
3. Complete the onboarding (3 slides)
4. Tap **+** to scan a receipt
5. Confirm/edit vendor, date, amount, and category
6. Save to your collection
7. View Reports tab for monthly spending breakdown
8. Export to CSV or connect QuickBooks/Xero in Settings

## Tech

Single HTML file with vanilla JS, localStorage persistence, responsive mobile-first design.
